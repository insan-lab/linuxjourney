## Lesson Content

¡Felicidades por completar las lecciones fundamentales de tu viaje en Linux! Has cubierto los conceptos básicos esenciales de Linux, y ahora es el momento de aprender a finalizar tu sesión correctamente. Salir de la shell de Linux es un paso final simple pero importante.

### El Comando Exit

La forma más común de finalizar una sesión de shell es con el comando `exit`. Cuando escribes `exit` y presionas Enter, el proceso de shell actual termina. Este es un comando universal que funciona en prácticamente cualquier entorno de shell, lo que lo convierte en una parte fundamental de cualquier tutorial de Linux para principiantes.

```bash
exit
```

### El Comando Logout

Otro comando que puedes usar para salir de la terminal es `logout`. Este comando está diseñado específicamente para terminar una shell de inicio de sesión. Aunque en muchos sistemas modernos se comporta de manera similar a `exit`, es una buena práctica conocer ambos comandos.

```bash
logout
```

### Cerrar la Ventana de la Terminal

Si estás trabajando dentro de una interfaz gráfica de usuario (GUI), también tienes la opción de simplemente cerrar la ventana de la terminal. Esta acción generalmente envía una señal que termina la sesión de shell que se ejecuta en su interior, proporcionando una forma rápida de realizar una salida de terminal.
